## Requirments
- dotnet core 2.2
- editor of your choice
- gpg key added to accepted contributors


## Installation
```console
$ git clone git@github.com:smbc-digital/verint-service.git
$ cd verint-service/src
$ git-crypt unlock
$ dotnet run
```
